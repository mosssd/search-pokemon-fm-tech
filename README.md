# search-pokemon-fm-tech

Pokemon search app built with **Next.js + GraphQL (Apollo)**.

## Live Demo

[https://search-pokemon-fm-tech-coral.vercel.app/](https://search-pokemon-fm-tech-coral.vercel.app/)

## What it does

* Search Pokemon by name (URL synced)
* Show details, attacks, evolutions
* Click evolution to search it

## Stack

Next.js • Apollo GraphQL • Tailwind

## Run locally

```bash
npm install
npm run dev
```
